 # Hungry? - The Happy Hour Pairing Generator
>### Project 1 UNC Coding Bootcamp - Interactive Front-End Project <br><br>
> This application is designed to append food and beverage pairings based on user input to curate the happiest of hours. Designed with two API's, theMealDB and theCocktailDB, this application collects user data via a drop down list, queries the database for food and drink options within the selected criteria and renders a randomized reccomendation to the user. <br><br>

## Usage

  * Click the start button
  * Select your alcohol of choice from the drop-down list
  * Choose a food category
  * View your results!
